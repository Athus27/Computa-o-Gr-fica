# Fluxo Git e Commits (DAP)

- **Fluxo de Trabalho**
```
1. git pull origin main

2. git checkout -b nome-da-branch

3. git add .

4. git commit -m "tipo: descricao"

5. git push origin nome-da-branch

6. git checkout main

7. git merge nome-da-branch`
```

# Padrao de Commits (Conventional Commits)
```• feat: nova funcionalidade

• fix: correcao de erro

• docs: documentacao

• style: formatacao e estilo

• refactor: refatoracao de codigo

• chore: manutencao ou dependencias

Exemplo

git commit -m "feat: implementa sistema de busca"`
```
